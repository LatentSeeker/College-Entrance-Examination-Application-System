### 数据字段说明

完整CSV包含以下字段：

| 字段 | 说明 |
|------|------|
| 学校ID | 学校唯一标识 |
| 学校名称 | 学校全称 |
| 年份 | 录取年份（2025） |
| 录取批次 | 本科批/本科提前批等 |
| 招生类型 | 普通类/中外合作办学/国家专项等 |
| 专业组 | 专业组编号（如：专业组（501）） |
| 选科要求 | 首选物理/历史，再选科目要求 |
| 最低分 | 专业组最低录取分数 |
| 最低位次 | 专业组最低录取位次 |
| 专业名称 | 具体专业名称 |
| 专业最低分 | 该专业最低录取分数 |
| 专业最低位次 | 该专业最低录取位次 |

### 下一步操作建议

1. **查看数据**：用Excel打开 `output/高校分数线完整数据_20260624_125534.csv`

2. **数据筛选**：
   - 按学校筛选：查看特定高校的所有专业组
   - 按分数筛选：找出分数范围内的所有选项
   - 按选科筛选：根据自己的选科组合筛选

3. **继续抓取**：
   - 如需更多学校，运行 `python gaokao_scraper.py`
   - 修改脚本中的 `end_id` 参数扩大范围

4. **数据转换**：
   - 如有新的JSON数据，运行 `python convert_to_csv.py` 转换
